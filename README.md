## Chip 8 emulator/interpreter built with C++ 20 and SDL3

## Built With
* C++
* SDL3
* CMake

## How to run the emulator
* Git clone the repo into a directory of your choice.
* Navigate to 3rdparty/3rdparty.
* git pull SDL3 repo.
* Navigate to the root directory of the project and build.
* Load and run ROMs.
